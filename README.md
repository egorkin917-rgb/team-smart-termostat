# Климат-контроллер: умный термостат с расписаниями и реле

•	Климат-контроллер: умный термостат с расписаниями и реле.
• Автономное устройство на базе Raspberry Pi Pico для автоматического поддержания температуры в лаборатории. Термостат управляет нагревателем/вентиляцией по расписанию (день/ночь), имеет ручной режим и защиту от частых включений реле.
